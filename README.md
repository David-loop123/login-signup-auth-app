# Login & Signup Authentication App

A full-stack authentication system built with HTML, CSS, JavaScript, and Node.js.

This project is currently in **active development** and demonstrates real-world authentication flow with frontend validation and backend integration.

---

## Features

- Login & Sign-up pages
- Client-side form validation
- Inline error & success messages
- Password visibility toggle
- Dark / Light mode toggle
- Responsive UI
- Node.js + Express backend
- JSON-based user storage (development mode)

---

## Tech Stack

**Frontend**

- HTML5
- CSS3
- JavaScript (ES6+)

**Backend**

- Node.js
- Express
- File-based storage (`users.json`)
